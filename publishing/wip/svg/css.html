Whenever possible, define style properties for graphical components in a separate style sheet. Cleanly separating styling allows users to apply their own custom styles if they have difficulty with the default appearance.

Avoid adding the style attribute directly to graphical components, as it can interfere with the ability of a user to change the appearance.

Whenever embedding text, ensure there is a strong contrast between the prose and any background images so that the two don't blend together.

See the EPUB Style Sheets section of this guide for additional considerations.
